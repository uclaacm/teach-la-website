Developer Resources

For all new members of the developer team, we have compiled a variety of resources to help them gain experience with web development. Below is a list of all the major resources that are useful for new developers.

Templates

React Starter Template

We have created a template for a React micro-app that is free to use.

The template is useful for people trying to create small projects with React.

Browse the template to get started.

Learning Resources

Dev Team Intern Training (Winter '21)

For new interns for the dev team, we have compiled some notes and resources to help with getting integrated into the team.

The compilation is available on GitHub.

QWER Hacks

We have gathered a series of written notes, code demos, and video links originally ran at QWER Hacks.

These resources provide a lot of valuable information about general web development topics like React and Firebase.

The collection from 2021 and the collection from 2020 are available on GitHub.

Learning Labs Crash Course

For new members of the dev team, we have created a repository that contains all of the notes and videos from the Learning Labs Crash Course taught over the summer.

This series is a good starting point for poeple with little to no web development experience.

Browse the crash course on Github or the playlist on YouTube to get started.
