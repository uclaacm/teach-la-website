Everything you need to know about getting started with Teach LA's dev team.

Learn how to set up your development environment for our website or projects, add yourself to the teams page, add or remove classes/resources, and more! No previous coding experience required!

Developer Set Up

Follow along with our in-depth setup walkthrough to get familiar with your shell and GitHub, which every person on the dev team needs!

...or, if you're already a command-line guru and assuming you have git and Ruby: 

  
$ gem install bundler
$ git clone https://github.com/uclaacm/teach-la-website.git
$ cd teach-la-website
$ bundle
$ bundle exec jeykll serve
