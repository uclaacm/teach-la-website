ACM Teach LA

making coding accessible. fun. empowering. approachable. easy.

ACM Teach LA pairs UCLA students with Los Angeles schools to empower K-12 students with the ability to code by offering free computer science classes.

Let's Work Together! Learn More

Virtual Learning

Following the latest updates of the COVID-19 pandemic, Teach LA has shifted most of its instruction back to in person. Virtual exceptions include our Python courses. As our partner schools progress to in person learning, we will continue working to best serve our students.

How does it work?

Teach LA’s flagship program teaches weekly classes with a wide variety of topics, ranging from introductory Python or web development to applications such as machine learning and data science. Each class balances instruction and interactivity with the support of our trained teachers.
